# Job Application Tracker (Career Dashboard)

A simple Next.js + TypeScript + Tailwind app to track job applications with search, filters, and a dashboard summary.
Data is stored locally using browser LocalStorage (no backend yet).

## Features
- Add / edit / delete job applications
- Dashboard stats (Total, Applied, Interviewing, Offers)
- Search by company or role
- Filter by status
- Sort by recently updated
- Application details page

## Tech Stack
- Next.js (App Router)
- TypeScript
- Tailwind CSS
- LocalStorage (persistence)

## Run locally
```bash
npm install
npm run dev
